pk.chart Class

Defined in: src/chart/chart.js:2
Module: pocketknife

Create a new chart component

HTML

<div id='chart'></div>

Javascript:

pk.chart({
                            element: document.getElementById('chart'),
                            type:'pie',
                            data: [
                                {category:"cat",series1:8, series2:2},
                                {category:"pig",series1:2, series2:6},
                                {category:"cow",series1:9, series2:9},
                                {category:"bird",series1:5, series2:2},
                                {category:"dog",series1:2, series2:7},
                                {category:"emu",series1:6, series2:3},
                                {category:"hamster",series1:2, series2:3}
                            ],
                            center:20,
                            legend:true,
                            axis:{
                                x:'category'
                            },
                            series: ['series1', 'series2']
                        });

Parameters:

  • type String

    Chart type to create, pie (use to create donut charts), line, scatter (use to create bubble charts), area, column or bar

  • [center=0] Number optional

    If chart type pie, the inner radius to create a donut chart, expressed as a percentage of diameter (2*r)

  • data Object

    Object array of data to use for the chart

  • axis Object

    Object keys in data to use for x and y axes

    • x String

      Object keys in data to use for x axis

    • [y] String optional

      Object keys in data to use for y axis, marked optional as not required for pie charts

    • [r=false] String optional

      String denoting object key in data to use for point radius, can be set to number

  • series Array

    Object keys in data used for series designation - only an Array for pie charts, string for all other types

  • [margin] Object optional

    Object of top, right, left and bottom margin amounts in pixels. Defaults to 20,20,50,20

  • [gridlines=true] Boolean optional

    Show gridlines for chart types other than pie

  • [colors] Object optional

    Object key value pairs where the key is a series name, the value is the value to use

  • [legend] Function optional

    Custom function responsible for building chart legend, defaults to default constructor if not passed

  • [tooltip=false] Function optional

    Custom function responsible for building chart legend, defaults to default constructor if not passed
