pk.color Class

Defined in: src/utils/color.js:2
Module: pocketknife

Utility class for converting color types and generating color schemes.

For color conversion methods, where the expected parameter is an array e.g [0,0,0], the method can also be passed a comma separated value string, e.g. '0,0,0', from which an array will automatically be extrapolated.

algorithmic

(
  • options
)
Array

Defined in src/utils/color.js:577

Generate algorithmic color palette

var palette=pk.color.algorithmic ({hex:'#FF0000'});
// palette = ["#ff0000", "#00ff00", "#0000ff"]         

var palette=pk.color.algorithmic ({hex:'#FF0000', count: 5});
// palette = ["#ff0000", "#ccff00", "#00ff66", "#0066ff", "#cc00ff"]              

var palette=pk.color.algorithmic ({hex:'#FF0000', scope: 100, count:4});
// palette = ["#ff00d4", "#ff0047", "#ff4700", "#ffd500"]

var palette=pk.color.algorithmic ({hex:'#FF0000', type:'saturation', scope:50});
// palette = ["#ffffff", "#ffbfbf", "#ff8080"]

Parameters:

  • options Object

    Algorithmic color transformation options

    • hex String

      HEX color string

    • [count=3] Number optional

      Number of colors to produce

    • [type=hue] String optional

      Component to calculate on, either hue, saturation or value

    • [scope=360] Number optional

      Number of degrees of rotation to consider

    • [rotation=0] Number optional

      Number of degrees of rotation to calculate from

Returns:

Array:

Returns array of HEX color strings

Generate (palette of) complementary color(s) from passed HEX color string

Generate a color from a percentage point between two colors

// color = "#f2e600";

Parameters:

  • perc Number

    Percentage between first and second colors to generate color point from expressed as value between zero and one

  • [hex1=#ff0000] String optional

    Starting color HEX value defaults to #ff0000 red so 0% is red

  • [hex2=#27ae62] String optional

    Starting color HEX value defaults to #1ae500 green so 100% is green
