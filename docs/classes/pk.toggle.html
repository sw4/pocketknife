pk.toggle Class

Defined in: src/toggle/toggle.js:2
Module: pocketknife

Create a new toggle control

HTML:

<input id='toggle' />

Javascript:

pk.toggle({
    element: document.getElementById('toggle'),
    options: [{
        value: true,
        name: 'yes'
    }, {
        value: false,
        name: 'no'
    }, {
        value: 'other',
        name: 'other'
    }],
    value: false
});
